# Multica AI — Landing Page

เว็บไซต์ Landing Page สไตล์ AI/SaaS โมเดิร์น สร้างด้วย HTML / CSS / JavaScript ล้วน (ไม่มี framework, ไม่ต้อง build)

## ✨ Features
- ดีไซน์โมเดิร์น ธีมมืด พร้อม gradient และ animated background orbs
- Responsive เต็มรูปแบบ (มือถือ / แท็บเล็ต / เดสก์ท็อป)
- Reveal-on-scroll animation, animated stat counters
- เมนูมือถือ (hamburger), FAQ accordion, ปุ่ม back-to-top
- ฟอร์มสมัครพร้อม validation อีเมล
- รองรับ `prefers-reduced-motion`

## 📁 โครงสร้าง
```
project01-website/
├── index.html
├── css/style.css
├── js/main.js
└── README.md
```

## 🚀 วิธีรัน
เปิดไฟล์ `index.html` ด้วยเบราว์เซอร์ได้เลย หรือรันผ่าน local server:

```bash
# Python
python -m http.server 8000
# แล้วเปิด http://localhost:8000
```
